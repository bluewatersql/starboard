# scripts

Development and build utility scripts. Not shipped in packages.

## Key Scripts
- `generate_diagrams.py` -- Mermaid diagram generation for docs
- `export_api_schemas.py` -- API schema export for contract tests
- `debug_browser.py` -- Playwright Chrome launcher for frontend debugging
- `build_rag_vector_store.py` -- RAG vector store builder
- `analyze_test_coverage.py` -- Test coverage analysis
- `validate_doc_links.py` -- Documentation link validation
- `validate_prompt_alignment.py` -- Prompt alignment checks
- `check_doc_staleness.py` -- Documentation freshness checks
- `generate_test_template.py` -- Test boilerplate generator
- `generate_types.py` -- Type generation utilities
