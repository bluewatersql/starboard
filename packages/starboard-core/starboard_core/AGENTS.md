# starboard-core

Pure domain models, prompts, shared types. Zero I/O dependencies.

## Key Modules
- `models/` -- Domain models (Conversation, Episode, Fact, UserProfile)
- `domain/` -- Core domain logic and value objects
- `foundations/` -- Base classes and shared abstractions
- `ports/` -- Port interfaces for dependency inversion
- `rag/` -- RAG (retrieval-augmented generation) domain types
- `repositories/` -- Repository interface definitions
