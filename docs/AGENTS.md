# docs

MkDocs documentation site. Build with `make docs`, serve with `make docs-serve`.

## Structure
- `index.md` -- Landing page
- `overview/` -- Product overview and agent catalog
- `user-guide/` -- End-user workflows and reference
- `agents/` -- Agent documentation (domain agents, framework)
- `architecture/` -- System architecture and design docs
- `developer/` -- Developer guide, standards, agent/tool development
- `tools/` -- Tool catalog and development guides
- `admin/` -- State backends and monitoring
- `integration/` -- API quickstart, SSE streaming, package integration
- `contracts/` -- Agent output consumer contracts
- `runbooks/` -- Incident response and operations templates
- `guides/` -- Getting started, common tasks, contributing, FAQ
- `packages/` -- Per-package documentation
