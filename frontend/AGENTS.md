# Frontend

Next.js 16 web UI with React 19, Material UI v7, and Tailwind CSS v4.

## Key Directories
- `app/` — Next.js App Router pages and layouts
- `components/chat/` — Chat UI (MessageList, ChatContainer, MessageInput)
- `lib/hooks/` — Custom hooks (useSSE, useConversation)
- `lib/store/` — Zustand stores (conversationStore, messageStore)
- `lib/validation/` — Zod schemas for API contracts and SSE events
